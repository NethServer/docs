Administrator Manual¶

NethServer es un sistema operativo diseñado para oficinas pequeñas y medianas empresas. Es muy sencillo, seguro y flexible.

Acerca de:

  • Sitio Oficial: http://www.nethserver.org
  • Rastreador de errores: http://dev.nethserver.org
  • Twitter: @NethServer
  • Codigo fuente: http://code.nethserver.org
  • Lista de correo: nethserver@googlegroups.com
  • Arvhivo ML: Google Groups
  • IRC: #nethserver on freenode.net
